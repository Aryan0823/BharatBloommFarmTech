## 🌾 Project Overview

A modern, minimalistic single-page website for **BharatBloomm FarmTech Private Limited** — India's leading AgriTech company specializing in battery-operated agricultural tools powered by advanced lithium technology.

This website showcases the company's innovative multipurpose farming equipment designed specifically for modern Indian agriculture, combining cutting-edge technology with sustainable farming practices.

## 🚀 Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Custom styling with:
  - Glassmorphism effects (`backdrop-filter: blur(16px)`)
  - CSS Grid & Flexbox for layouts
  - CSS animations and transitions
  - Responsive media queries
- **Vanilla JavaScript** — Interactive features:
  - Scroll-based navbar transformation
  - Intersection Observer for scroll animations
  - Smooth anchor link scrolling
  - Form submission handling
  - Mobile menu toggle
- **Google Fonts** — Inter (body) + Playfair Display (headings)
- **Font Awesome 6.4.0** — Icons throughout the site

## 🎨 Design System

### Color Palette
- **Primary Green:** `#4CAF50`
- **Dark Green:** `#2E7D32`
- **Light Green:** `#A5D6A7`
- **Text Dark:** `#1B1B1B`
- **Text Muted:** `#555555`
- **White Background:** `#FFFFFF`
- **Footer Dark:** `#1B4332`

### Typography
- **Headings:** Playfair Display (600-800 weight)
- **Body Text:** Inter (300-800 weight)
- **Base Size:** 16px with 1.6 line-height

### Border Radius
- All cards and buttons: `20px` (rounded)
- Small elements: `12px-16px`
- Circular elements: `50%`

## 🌐 Deployment

This is a static single-page website ready for deployment on any web server or hosting platform:

- **No backend dependencies** — Pure HTML/CSS/JavaScript
- **No build process required** — Ready to deploy as-is
- **Compatible with:** GitHub Pages, Netlify, Vercel, AWS S3, Azure Static Web Apps, or any traditional web hosting

### Deployment Steps:
1. Upload `index.html` to your web server root directory
2. Ensure all image URLs are accessible (currently using external CDN links)
3. Configure domain and SSL certificate
4. Test responsiveness across devices
